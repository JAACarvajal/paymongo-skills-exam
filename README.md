# Paymongo Skills Exam

### Techonology Used
- PHP 8.1
- Composer 2.5.8
- Carbon 2.70

### Installation
1. Install dependencies
    ```
    composer install
    ```
2. Run script
    ```
    php index.php
    ```

### Implementation
- Creational Design Pattern
    - Factory Method or Constructor Pattern
- Dependency Injection
- Classes
    - ParkingSystem
    - ParkingSlot
    - Vehicle
